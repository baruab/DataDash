DataDash

DashDash is an open-source initiative to provide developers and business users with a tool to upload data in multiple file formats, combine, merge and transform content to help represent the information in a more organized and easy to interpret fashion.

We all use different software applications in our daily work life and often are bombarded with business reports from various sources, which we have to review or monitor for changes for daily business needs. This project is an attempt to make this repetitive process for the business users easy to access and extend on the functionality using open-source, modular programming techniques.

DataDash is a R Shiny app which  is based on a package based software stack for building interactive web applications. For ex: Shinydashboard package provides a convenient way to create a bootstrap interface to transform, clean and join data using the robust R package support for statistical analysis. The Shiny reactive programming paradigm allows developers to specify a graph of dependencies so that when an input changes, all related outputs are automatically updated. DataDash tool exploits this reactive feature to simulate interactive data modeling behavior and representing the outcome as plots and graphs..

Special emphasis is given during the DataDash tool implementation to source code organization using modules. Module encapsulates the business functionality using clearly defined dependencies. It can then easily placed as a sub unit within a R package. The long term vision of this project is to build on such functionalities by creating multiple modules, to be eventually clubbed together as a new ‘datadash’ R package.
